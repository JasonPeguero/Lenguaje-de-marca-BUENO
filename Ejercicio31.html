Introducción

SQL, Structure Query Language(Lenguaje de Consulta Estrcuturado) es un lenguaje de programación para trabajar con base de datos relacionales como MySQL, Oracle, etc. MySQL es un interpretador de SQL, es un servidor de base de datos.
